# HOTEL GUJRAT INN — Responsive Hotel Website

A modern and responsive hotel website built for **HOTEL GUJRAT INN, Rajnandgaon**. The site showcases rooms, amenities, gallery images, and guest testimonials with a clean layout, easy navigation, and seamless booking/contact options.

## Features

* Fully responsive design
* Sticky header with clear navigation
* Full-width hero sections and image carousel
* Uniform gallery grid for consistent visuals
* Rooms and amenities sections with CTA buttons
* Direct booking link and WhatsApp quick contact
* Integrated Tawk.to live chat
* Google Analytics and Google Translate support
* SEO-friendly meta tags and social previews

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Bootstrap (optional responsive components)
